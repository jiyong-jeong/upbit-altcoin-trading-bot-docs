# Upbit + Binance 자동 매매 봇 — 알트코인 포트폴리오 전략 노트

> Upbit / Binance 거래소를 동시 운용하는 개인용 암호화폐 자동 매매 봇의 구조와 전략 분류 정리. Python + pyupbit/ccxt 기반의 학습/개인 사용 목적 봇 패턴의 관찰자 시점 문서.

**카테고리:** 개인용 알트코인 자동 매매 봇
**스택:** Python, pyupbit (Upbit API), ccxt (Binance API), Line Messaging API
**눈여겨볼 지점:**
- **두 거래소 동시 운용** — Upbit (현물 KRW 마켓) + Binance (현물/선물 USDT 마켓)
- **포트폴리오 분류 전략** — 비트코인 / 이더리움 / 알트 베스트 (장기) / 알트 단타 A/B 타입
- **단타 A/B 분리** — 상승장에서만 매수하는 A타입, 하락장에서도 매수하는 B타입
- API 키 **AES 류 자체 암복호화** + 별도 키 파일 — 디스크 평문 보관 회피
